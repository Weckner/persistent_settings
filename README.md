# Persistent Settings

A World of Warcraft addon that lets you inherit settings from your main character to all alts.

## Status

This is a hobby project and work-in-progress (WIP), created to help a friend.

## Features

- Designate a "main" character
- Select which settings should be inherited by alts
- Automatically apply settings to new characters
- Manual apply option for existing characters

## Installation

1. Download or clone this repository
2. Copy the `persistent_settings` folder to your WoW addons directory:
   ```
   World of Warcraft\_retail_\Interface\AddOns\
   ```
3. Restart WoW or type `/reload` in-game

## Usage

- `/ps` - Open the settings window
- `/ps main` - Set current character as main
- `/ps apply` - Apply settings from main to current character
- `/ps status` - Show current main/character info
- `/ps help` - Show all commands
